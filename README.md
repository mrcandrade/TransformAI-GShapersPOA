## 🌟 Sobre o Projeto

**TransformAI** é uma iniciativa educacional do [Global Shapers Community — Hub Porto Alegre](https://www.globalshapers.org/) que leva o conhecimento sobre Inteligência Artificial para escolas da rede pública, democratizando o acesso à tecnologia de ponta.

Este repositório contém o material completo do **projeto piloto**, desenvolvido para capacitar estudantes do ensino médio a entender, questionar e utilizar IA de forma crítica e ética.

> 💡 *"A IA não deve ser privilégio de poucos. Queremos que cada estudante entenda como ela funciona, suas limitações e seu potencial de transformação."*

---

## 🎯 Objetivo

Preparar jovens para o futuro digital, ensinando:
- **Como a IA "pensa"** — tokens, probabilidades, temperatura
- **Como conversar com IA** — prompt engineering na prática
- **Quando desconfiar** — alucinações e verificação de fatos
- **Responsabilidade** — viés, ética e impacto social

## 🚀 Como Usar

### Pré-requisitos
- Conta no [Google Colab](https://colab.research.google.com/) (gratuito)
- Chave da API Anthropic ([console.anthropic.com](https://console.anthropic.com))

### Passo a Passo

1. **Abra o notebook no Colab**
   - Clique no notebook desejado → "Open in Colab"

2. **Configure sua chave de API**
   ```python
   os.environ["ANTHROPIC_API_KEY"] = "sua-chave-aqui"
   ```

3. **Execute as células em ordem**
   - Use `Shift + Enter` para rodar cada célula

4. **Complete os desafios**
   - Procure por `# TODO` no código
   - Preencha os arquivos `respostas-XX.md`
